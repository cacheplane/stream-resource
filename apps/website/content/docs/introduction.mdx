# Introduction

StreamResource is the Angular streaming library for LangChain and LangGraph.

## Installation

```bash
npm install stream-resource
```

## Quick start

```typescript
import { streamResource } from 'stream-resource';

const chat = streamResource({
  assistantId: 'chat_agent',
  apiUrl: 'http://localhost:2024',
});
```

More documentation is generated automatically — run `npm run generate-docs` to populate this directory.
