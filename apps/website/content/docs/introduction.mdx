# Introduction

Agent is the Angular streaming library for LangChain and LangGraph.

## Installation

```bash
npm install angular
```

## Quick start

```typescript
import { agent } from 'angular';

const chat = agent({
  assistantId: 'chat_agent',
  apiUrl: 'http://localhost:2024',
});
```

More documentation is generated automatically — run `npm run generate-docs` to populate this directory.
